Test management

Create test plans and cases, track progress and assign work across multiple teams. Perform peer reviews.

Process organization

Workflow wizards guide you through requirements gathering, user story definition and test planning.

Testing telemetry

Gives you instant insight into your testing. All the vital info for planning and status reports before release!

Integration

Integrated bug reporting with Bugzilla, JIRA, GitHub, GitLab & more. Additional workflow integration via GitHub App.

Test runner plugins

Use Kiwi TCMS plugins for TAP, JUnit, TestNG, py.test and other popular testing frameworks to collect automation results!

Additional Information for Kiwi TCMS Subscriptions

  • Subscriptions are renewed automatically each month. You may cancel at any time if you are not happy with the service provided
  • Cancelled subscriptions remain active until the period they have been paid for - usually 1 month after the last billing transaction
  • Professional support is independent of hosting! You may chose a Private Tenant subscription and still decide to host Kiwi TCMS internally
  • Subscriptions cover Kiwi TCMS, Kiwi TCMS Enterprise and all add-ons published under https://github.com/kiwitcms/
  • All subscriptions are stackable giving you hosting/support flexibility
  • Devel engagement means that our team will start working on GitHub issues within the specified time frame after you contact support about said issues
  • We will try our best to estimate how long a particular task will take, however Devel engagement is not a completion deadline
  • Use add-on services for anything which isn't a support question, a bug, a feature request or related to managed hosting
  • If it's not a system or repository that we control then it needs additional services. Don't hesitate to talk to us!
